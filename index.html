What we're building

Cellframe is a TypeScript-first CLI and SDK that gives FunC and Tact developers a deterministic, in-process TVM sandbox with per-opcode gas accounting, BOC-level state snapshots, and a mainnet-fork mode that pulls live shardchain state via TonCenter v3 for regression replay. Tests are written in a Jest-compatible syntax so existing tooling habits carry over, while the runner natively models TON's async bounce-message semantics and split-brain sharding — patterns that silently break when Ethereum-ported test frameworks are applied to TON contracts. A built-in flamegraph reporter diffs gas consumption across commits and outputs a CI artifact, closing the feedback loop between a code change and its on-chain cost impact. The `cellframe init` scaffold generates a complete FunC or Tact project with sample tests, a GitHub Actions workflow, and a Blueprint migration shim, so teams can adopt it without restructuring an existing repo.
